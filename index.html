Расчёт фазового состояния по уравнению Пенга — Робинсона

Инструмент выполняет расчёт вспышки (isothermal flash) для выбранной смеси, определяет состав фаз, плотность, вязкость, теплопроводность и теплоёмкость каждой фазы на основе термодинамических и транспортных корреляций промышленного уровня точности.

Исходные данные

Компонентный состав (мольные доли)

Значения концентраций нормируются автоматически. Для расчёта фазовых свойств используется точное уравнение состояния Пенга — Робинсона с итерационным уточнением коэффициентов равновесия по фугитивностям.
